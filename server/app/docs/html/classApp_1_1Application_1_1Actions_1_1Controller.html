Retorna una respuesta HTTP con el contenido y el código de estado especificados.

Este método configura la respuesta HTTP con el código de estado dado, establece el tipo de contenido a JSON y escribe el cuerpo de la respuesta con los datos proporcionados.

Parameters
$responseLa respuesta HTTP que se configurará.
$resLos datos que se incluirán en el cuerpo de la respuesta.
$statusCodeEl código de estado HTTP (por defecto es 200).
Returns
$responseStatus devuelve el status configurado.

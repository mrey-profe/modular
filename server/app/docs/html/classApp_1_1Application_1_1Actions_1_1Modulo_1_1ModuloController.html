Lista los módulos según el ID de departamento.

Este método recibe un ID de departamento y retorna una lista de módulos asociados a ese departamento. Si no se proporciona un ID, se listan todos los módulos.

Parameters
$requestLa solicitud HTTP.
$responseLa respuesta HTTP que se enviará.
$argsArgumentos adicionales que contienen el 'id' del departamento.
Returns
$response devuelve la lista de módulos

\hypertarget{index_descripcion}{}\doxysection{Descripción General\+:}\label{index_descripcion}
Este proyecto es una aplicación diseñada para facilitar la gestión de profesores, módulos y modelos creados por los profesores en el ámbito educativo. Su propósito es simplificar y agilizar el proceso de asignación de materias para el próximo curso académico. Ofrece una interfaz intuitiva y eficiente para que los profesores elijan las materias que desean impartir, mientras que el administrador puede gestionar la asignación de horarios y materias de manera centralizada.\hypertarget{index_funcionalidades}{}\doxysection{Funcionalidades Principales\+:}\label{index_funcionalidades}

\begin{DoxyItemize}
\item Gestión de usuarios (profesores y administradores).
\item Creación y administración de módulos.
\item Soporte CRUD (Crear, Leer, Actualizar, Eliminar) para módulos y modelos. Aunque por ahora solo C
\item Interfaz de usuario intuitiva y fácil de usar.
\item Implementación segura y escalable utilizando Docker.
\end{DoxyItemize}\hypertarget{index_estructura}{}\doxysection{Estructura del Proyecto\+:}\label{index_estructura}
El proyecto está organizado en los siguientes módulos\+:


\begin{DoxyItemize}
\item Backend\+: Contiene la lógica del servidor y las API REST.
\item Frontend\+: Contiene la interfaz de usuario y las vistas correspondientes.
\item Base de datos\+: Define los modelos de datos y las migraciones.
\item Despliegue\+: Archivos y configuraciones para el despliegue utilizando Docker.
